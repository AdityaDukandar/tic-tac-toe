# 🕹️ XOX ULTRA INSANE

Welcome to **XOX ULTRA INSANE**, a high-energy, neon-drenched, retro-arcade Tic-Tac-Toe game styled with gorgeous procedural pixel art, vintage CRT effects, screenshake, particle explosions, and synthesized 8-bit sound effects. 

Designed to run beautifully in any browser, it includes an unbeatable **Minimax "Insane AI"** that taunts you as it calculates your doom.

---

## 🎨 Immersive Retro Visuals & Audio
* **CRT Screen Emulation:** Scanlines, screen flicker, radial vignette shadow, and pixelated glow overlays mimic vintage arcade monitor hardware.
* **Procedural Pixel Portraits:** Zero image assets! 4 unique fighter avatars (`CYBER_PUNK`, `RETRO_RED`, `GLITCH_VOID`, `GOLD_MECH`) are procedurally generated and rendered pixel-perfect using nested arrays and custom `<canvas>` setups.
* **Game Feel & "Juice":** Intense screenshake on critical plays, dynamic particle explosions (sparks, glitch blocks, plasma trails), and white-hot laser beams bridging winning lines.
* **Web Audio Synthesis:** Sound effects are generated in real-time using custom synthesizers and noise-buffers via the native Web Audio API (no external MP3/WAV assets required!).

---

## 🎮 Game Modes
1. **VS AI OPPONENT:** 
   * **Easy:** Prone to blunders, perfect for casual matches.
   * **Hard:** Plays strategically, capitalizing on your mistakes.
   * **Insane:** Runs a flawless, depth-optimized Minimax AI. Unbeatable. Accompanied by unhinged, sarcastic digital terminal taunts!
2. **LOCAL 2-PLAYER:** Pass-and-play local multiplayer mode.

## 🚀 Instant Deployment (Vercel)

This repository is pre-configured for instant static hosting on **Vercel** with a clean URL routing and custom secure headers configured in `vercel.json`.

### Deploying using Vercel CLI:
```bash
# Install Vercel CLI globally (if not installed)
npm install -g vercel

# Deploy instantly
vercel
```

---

## 💻 Running Locally

Simply open the `index.html` file in any modern web browser to play! 

No compilers, local servers, or package installations are required—it is written in clean, modern vanilla ES6 JavaScript, HTML5, and CSS3.

```bash
# Clone the repository
git clone <your-repository-url>

# Open in browser
open index.html
```
